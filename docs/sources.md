# Sources

このリポジトリの原則は、ユーザーのメモと以下の公開資料をもとに、AIエージェント用の実践的な判断基準として再構成している。

## 公式ページ

- [元週刊少年ジャンプ編集者が漫画家から学んだことを書いていく](https://www.shonenjump.com/p/sp/2019/saito_blog/index.html)
- [ジャンプの漫画学校](https://jump-manga-school.hatenablog.com/)

## 取り込み方

- 原文の転載ではなく、レビュー時に使える判断基準へ要約・再構成する。
- 著者個人の経験にもとづく技術論として扱い、ジャンプ公式の唯一解として扱わない。
- 作品、媒体、作者、制作体制に応じて、どの原則を強く使うかを変える。
- AI漫画制作では、絵の生成品質だけでなく、読みやすさ、情報順序、表情、手の演技、位置関係、吹き出し導線を重視する。
