# X広告カルーセル漫画の原則

`format: "x-carousel"` のreview判断基準。公式仕様は2026-06-11にX Ads公式ヘルプで検証済み。

## 検証済み公式仕様（出典: https://business.x.com/ja/help/campaign-setup/creative-ad-specifications）

- 画像カルーセル: アスペクト比 **1:1 または 1.91:1**。1カルーセル内は同一比のみ。本ハーネスは1:1を採用
- 推奨サイズ: 1:1 は 800×800px 以上（specの推奨canvasは1080×1080）
- スライド枚数: **2〜6枚**
- 画像: PNG/JPEG、5MB以内
- ポスト本文: 280 weighted chars（CJKは1字=2。日本語は実質全角140字）
- 注意: 「単独画像広告で4:5/2:3/9:16が使える」という記事はオーガニック投稿仕様との混同。広告は1:1/1.91:1が基本

## カード構成原則（hook → body → cta）

1. **1枚目（hook）がすべて**: タイムラインで止まらなければ2枚目以降は存在しないのと同じ。
   大きい顔・問いかけ・あるある・違和感のどれかで1秒以内に引く。商品要素はゼロにする
2. **中間（body）は読み物**: ドラマを進め、次のカードへスワイプさせる「ヒキ」をカード末尾に置く。
   本誌のページめくりと同じ構造が、カルーセルではスワイプに対応する
3. **最終枚（cta）だけが広告**: 商品名・誘導・リンク誘導は最終カードに集約。
   途中に広告色が漏れると読了率が落ちる（読了がCVの前段）

## 広告色の抑制（事例パターン）

- グーネット「車Lv.1の僕と車Lv.100のギャル」: 1P漫画の連載風配信。広告色ゼロの読み物として
  成立させてエンゲージメントを取った
- freee「#株式会社もふもふ」: キャラ駆動（動物キャラ）で企業色を回避。愛嬌が抵抗感を消す
- comico: 既存作品の名場面カルーセル。ストーリーの連続性で「次へ」を引き出す
- 共通項: **広告ではなくエンタメとして読まれてから、最後に1回だけ売る**

## モバイル可読性

- 文字は大きめ・太め・高コントラスト。タイムラインのサムネイルサイズで読めない文字は存在しない文字
- 1カード1見せ場。カード内コマは1〜3が目安（lintは4超でwarn）
- カード文字量はソフト80字・ハード120字（lint連動）。スワイプの指が止まっている時間内に読み切れる量
- カード内は右→左の漫画読み、カード送りは左→右スワイプ。混在するので、コマ割りで読み順が
  曖昧になる構図（横並び問答など）は本誌以上に避ける

## reviewで必ず見るもの

1. 1枚目: スクロールが止まる絵か。顔は大きいか。問い・違和感・共感のどれが引きか言えるか
2. 各カード末尾: 次へスワイプさせるヒキがあるか
3. 広告色: 最終枚以外に商品名・宣伝文句が漏れていないか
4. 最終枚: CTAが1つに絞れているか（フォロー or リンク、両方は欲張り）
5. 文字量: スマホで読める量か。絵で語れるセリフを削れないか
