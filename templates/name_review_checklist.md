# ネームレビュー・チェックリスト

`name_v2.md` 完成後、**storyboard.json に進む前に必ず**このチェックリストでレビューする。

成果物: `production/spec/name_review.md`（各話）

## 判定

- **ship**: storyboard 化へ進める
- **revise**: ネームを直してから再レビュー
- **blocked**: 脚本へ戻る（会話不足・サブテキスト不足）

## チェック項目

### 脚本との関係

- [ ] 脚本の `must_say` がネーム上で落ちていない
- [ ] 脚本にない説明セリフをネームが勝手に追加していない
- [ ] 脚本の `scene_blocks` 順序とネームのページ流れが一致

### ページ設計

- [ ] 16P = 起4 / 承4 / 転5 / 結3 の目安に沿っている
- [ ] 各ページにヒキがある
- [ ] 1ページの文字量が多すぎない（目安150字前後）
- [ ] 同ページに「うん」等の反復セリフが3回以上ない

### 文字容器（描き分け）

- [ ] 非セリフ行が caption / monologue / sfx に分類されている
- [ ] 俯瞰・過去形・状態叙述は **caption** としてネームに載っている
- [ ] monologue（雲形）を使う場合は理由が明記されている（本作は原則不使用）

### 読み順・生成安全

- [ ] 質問→応答が同一行の横並び2コマになっていない
- [ ] モチーフ（バッグ、皿、毛布等）の追跡ページが書かれている
