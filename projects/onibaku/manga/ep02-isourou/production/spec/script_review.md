- **会話劇の核が成立している。** 「受かりました」→誰も「出ていくはず」を言わない→「飯、食う？」で追及を回避、という第1話末からの流れが脚本段階で明確。
- **「うん」の儀式化**が S3 モンタージュと visual_motifs で設計されている。セリフだけでなく正座・毛布・玄関待ち等の reaction_beats がある。
- **must_say / must_not_say** が adaptation_design と一致。Instagram・iPhone・横領説明は第3話へ送っている。
- **caption vs monologue 方針**が脚本末尾で明示され、本作ルール（monologue 原則不使用）に沿っている。
- **emotional_delta**（一晩限りの延長 → 「家を探す」を本気で受け取らなくていい）が episode_premise と整合。

## 修正推奨

1. **S3 モンタージュ（2〜5日目）**の dialogue_ladder が表形式の抜粋のみ。ネーム化前に、各日の「まけおとの言い方の変化」（出ていく → すいません → 明日には → 言わない）を1行ずつ固定しておくと、儀式のリズムがブレにくい。
2. **P10 相当（米・食材）**のセリフは脚本に未記載。ネームで追加する場合は、脚本の S3 に「生活が入る」段として1行足すか、絵のみ（caption）に留めるかをネーム段階で決める。

## ネームへ渡す注意

- 「うん」は多いが**短い**。同ページ3回以上並べない（ネーム側ルールと一致させる）。
- 内見シーン（S5）は**追及しない沈黙**が主役。セリフは「今日、内見？」「用事があって」「うん」+ caption で足りる。
- 初任給ケーキは白い箱のモチーフとして P15〜16 で追跡する。
- monologue（雲形）は使わない。結果・俯瞰はすべて caption。
